Perform matrix-vector operation : y := alpha*A*x + beta*y where alpha and beta are scalars, x and y are n element vectors and A is an n by n symmetric band matrix, with k super-diagonals.

At the moment support only diagonal matrix i.e. element-wise vector multiplication

Parameters
[in]AVector of element on the matrix's diagonal
[in]alphaScaling factor
[in]xVector
[in]betaScaling factor
[in,out]yOutput
